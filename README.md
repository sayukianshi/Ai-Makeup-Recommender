
# 💖 AI Makeup Bestie 💄✨
An AI-powered beauty assistant that analyzes your selfie and generates personalized makeup recommendations in seconds.  
Designed with a soft pastel aesthetic and interactive UI to create a fun, engaging user experience.

## 🌸 Features
✨ Face detection using computer vision  
✨ Skin tone analysis for personalized results  
✨ Complete makeup guide (foundation, eyes, lips, blush, etc.)  
✨ Cute pastel-themed interface with animations  
✨ Real-time results from uploaded images  

## 🧠 How It Works
1. Upload a selfie 📸  
2. The app detects your face and analyzes skin tone  
3. AI generates customized makeup suggestions 💄  
4. Results are displayed in a clean, aesthetic format  

## 🛠️ Tech Stack
- Python  
- OpenCV (Face Detection)  
- Streamlit (Web App UI)  
- NumPy
